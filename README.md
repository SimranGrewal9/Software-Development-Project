Python Software Development Project

Overview

This project was completed over four days to practice Python programming, software design, automation, testing, and documentation.

Features

- Student Management System
- File Handling
- Search, Update, and Delete Records
- Input Validation
- Automated CSV Data Analysis
- Exception Handling and Logging
- Unit Testing

Technologies Used

- Python 3
- CSV Module
- Logging Module
- Object-Oriented Programming (OOP)
- Git & GitHub

How to Run

1. Clone the repository.
2. Open the project folder.
3. Run:
   python main.py
